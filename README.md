# Match And Merge

## Summary:
- Made for the raylib 6.x game jam using C++ + raylib (also my first c++ project)
- My first ever C++ project, as well as raylib
- Tests your colour knowledge with ten different colours
- Allows you to input rgb values to merge (match) the square on the screen with the background (perfect, partial or minimal merge)
- You get points (min 0, max 10) per colour depending on if you skip or merge (perfectly, partially or minimally)
- Once you've gone through all the colours (10), you will be able to see your total score


## How to Play?:
You can play the game in one of two ways:
- Windows: by downloading the executable from this repo
- Browser: on itch.io (link - will add later)

### Controls:
- You select a textbox (r, g or b) and then input a number 0 through 255 or in hexadecimal (still 0-255)
- You can change the input type (hexadecimal/decimal) by pressing on the button beside each textbox
- The skip button (only available when no merge) can be used to skip the current colour, giving you 0 points for that one
- The next button (only available when you have some sort of merge) lets you continue to the next colour, by doing so you are awarded points based on the merge type
